# Auth Fetch App

A simple application for authentication and fetching data from resources.

# How To Install

## Auth app

go to auth folder

pip3 install -r requirements.txt

make .env file
copy env.example content to your env file, use your own configuration

To run app:
python3 app.py

## Fetch app

go to fetch folder

go mod download

make .env file
copy env.example content to your env file, use your own configuration

To run app:
go run main.go
